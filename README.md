## Overview

ShadowOS is a local-first desktop environment that runs inside a browser tab. It provides a full desktop-like workspace — windows, dock, menu bar, built-in apps, and local-first persistence — without depending on a cloud desktop or traditional OS packaging.

## OpenCore

This is the Core (open source) edition. An Enterprise edition with additional connectors, managed hosting, and commercial support is maintained separately under the same organisation.
